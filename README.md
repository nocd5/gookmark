# gookmark
Bookmark for CLI

## Description

## Usage

```bash
$ gookmark add
$ gookmark add ..
$ gookmark list
$ gookmark edit
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/nocd5/gookmark
$ gookmark config ui.editor=vim
$ gookmark config core.linefeed=unix
```
`gookmark config ui.editor=your_editor`  
`gookmark config core.linefeed=[unix|dos]`
